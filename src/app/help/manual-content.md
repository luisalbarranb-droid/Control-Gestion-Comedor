
# Manual de Usuario - Control Comedor v1.0

Este manual detalla todas las funcionalidades del sistema de gestión inteligente para comedores industriales.

## 1. Dashboard (Panel Principal)
El Dashboard es tu centro de comando. Aquí verás:
- **Monto en Almacén**: Valorización total de tu inventario en tiempo real.
- **Eficiencia General**: Rendimiento basado en el cumplimiento de tareas.
- **Stock Crítico**: Alertas automáticas de ingredientes agotados o próximos a agotarse.
- **IA Prioritizer**: Tu asistente inteligente que sugiere qué tareas deben atenderse primero.

## 2. Gestión de Inventario
### 2.1 Artículos
- Permite crear, editar y eliminar ingredientes o suministros.
- **Stock Mínimo**: Configura el nivel de alerta para cada producto.
### 2.2 Entradas y Salidas (Trazabilidad)
- **Escaneo con IA**: Puedes subir una foto de una factura y la IA extraerá los datos automáticamente.
- **Historial de Movimientos**: Registro auditado de quién, qué y cuándo se movió mercadería.

## 3. RRHH y Asistencia
- **Registro de Asistencia**: Los empleados pueden marcar entrada y salida mediante QR con validación visual mejorada.
- **Registro Manual**: Soporte para registro de asistencia manual en caso de fallos técnicos.
- **Gestión de Contratos**: Creación, edición y generación de contratos automatizada con marcadores inteligentes.
- **Planificación Semanal**: Gestión de días libres y rotación de turnos.
- **Evaluaciones**: Módulo para calificar el desempeño mensual del personal.

## 4. Planificación de Menús
- Creación de menús por días y servicios (Desayuno, Almuerzo, Cena).
- **Recetario**: Vincula cada plato con ingredientes del inventario para el descuento automático de stock.

## 5. Auditoría y Seguridad
- **Logs de Auditoría**: El sistema registra cambios críticos en la base de datos para garantizar la transparencia.
- **Roles y Permisos**: Acceso restringido según el perfil del usuario (SuperAdmin, Admin, Común).

## 6. Soporte e Impresión
- Este manual puede ser impreso pulsando el botón **"Exportar a PDF"** en la sección de Ayuda de la aplicación.
